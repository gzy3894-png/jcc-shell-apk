# JCC Shell APK 2.5.1

**2.5.1 = 2.5.0 的用法 + 当前赛季新数据。**

## 你不用懂技术

1. 到 [Releases](https://github.com/gzy3894-png/jcc-shell-apk/releases) 下 **v2.5.1** 的 APK  
2. 手机有 **Root**  
3. 安装后打开 App → 按界面启动注入（会自己拉游戏）  
4. 进**大厅**等一会儿  
5. 点刷新牌库，或用文件管理器看：  
   `/data/user/0/com.tencent.jkchess/files/jcc_cardpool.txt`
